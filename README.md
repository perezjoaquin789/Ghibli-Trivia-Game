# Ghibli Trivia 🎬

Aplicación web desarrollada con Flask que consume una API pública de Studio Ghibli.

## Funcionalidades
- Juego de trivia con 10 preguntas
- Preguntas generadas dinámicamente
- Sistema de puntaje
- Selector de dificultad

## Tecnologías
- Python (Flask)
- HTML
- API pública (Ghibli)

## Cómo ejecutar
pip install flask requests
python ghibli_trivia/app.py
